A JS Framework for HTML5 Canvas

lightweight, efficient, with zero dependencies

iio.js is a javascript library that speeds the creation and deployment of HTML5 Canvas applications. The download includes full production and development source code for the most recent version of the framework.

Free & Open Source

The core platform is open source and free to use for any purpose.

Submit additions, questions, and bugs on Github. General inquires may be sent to iio interactive.
